## [1.0.0] — 2026-04-10

### Added

- Next.js 16 application with App Router, Bun runtime, Prisma ORM, and NextAuth.js v5 (GitHub OAuth)
- `llm-aggregator` microservice — HuggingFace API proxy with Redis caching (5-minute TTL)
- `user-analytics` microservice — event tracking and user preferences backed by PostgreSQL
- `model-widget` micro frontend — standalone Vite + React SPA served at `/widget/`
- nginx reverse proxy for local dev and production
- Full observability stack: Prometheus, Grafana, Grafana Tempo, Grafana Loki, OTel Collector, Promtail
- Docker Compose configurations for development and production
- Kubernetes manifests with Kustomize (`kubectl apply -k k8s/`)
- Terraform modules for AWS EKS, VPC, RDS PostgreSQL, and ElastiCache Redis
- Ansible playbooks for server provisioning and deployment automation
- Test suites: unit, integration, smoke, and k6 stress tests
- GitHub Actions workflows: CI, CD, Terraform plan/apply, nightly stress tests
- `Makefile` — unified developer task runner covering install, lint, test, build, push, k8s, Terraform, and Ansible targets
- `Jenkinsfile` — declarative Jenkins pipeline mirroring the GitHub Actions workflow
- Sentry integration for Next.js (server, edge, and browser)
- VS Code dev container configuration
